# PRD Design Audit

A Codex skill for interaction designers to audit PRDs and turn them into design-ready briefs.

Use it to:

- Review whether a PRD is ready for design.
- Identify product, UX, metric, data, permission, and commercial risks.
- Evaluate whether Success Metrics are measurable and useful for design decisions.
- Convert PRD content into a design-facing overview.
- Extract commercial/domain terms and maintain a reusable glossary.

## Installation

Copy this folder into your Codex skills directory:

```bash
cp -R prd-design-audit ~/.codex/skills/
```

## Usage

```text
Use $prd-design-audit to review this PRD for design risks, success metrics, and glossary terms.
```
